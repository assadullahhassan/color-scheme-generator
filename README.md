# Color Scheme Generator 🎨

A minimalist, responsive web application that allows users to generate professional color schemes based on a seed color and a selected mode. Built using Vanilla JavaScript and powered by **TheColorAPI**.

## 🖼️ Demo

[Project link demo](https://assadullahhassan.github.io/color-scheme-generator/)

[Project Screenshot/video](https://drive.google.com/drive/folders/16rkGxfzRCShGK8RXKfXliMik7Qaqjy4S?usp=sharing)


## 🚀 Features
* **Seed Color Selection**: Pick any starting color using the native HTML color picker.
* **Mode Selection**: Choose from multiple color theory modes including.
    * Monochrome.
    * Monochrome-dark.
    * Monochrome-light.
    * Analogic.
    * Complementary.
    * Analogic-complementary.
    * Triad.
    * Quad.
* **Dynamic Generation**: Fetches color data in real-time from [TheColorAPI](https://www.thecolorapi.com/).
* **Hex Code Display**: Automatically displays the Hex codes for each generated color for easy copying.
* **Copy to clipboard**: Easy copying Hex code by clicking on any color


## 🛠 Tech Stack
* **HTML5**: Semantic structure for the interface.
* **CSS3**: Custom styling and flexbox-based layout for the color bars.
* **JavaScript (ES6+)**: Async/Fetch API to handle data and DOM manipulation.
* **TheColorAPI**: Async/Fetch API to handle data and DOM manipulation.


## 📋 How It Works
* **Select a Color**: Click the color box in the top left to choose your base color.
* **Select a Mode**: Use the dropdown menu to choose a scheme type (e.g., Monochrome).
* **Generate**: Click the "**Get color scheme**" button.
* **Result**: The app sends a request to the API and renders five distinct color bars with their corresponding Hex values.
